基于PRD需求设计的销售端详细测试用例，覆盖新建报价、历史管理、移动适配等核心功能。

**测试步骤**:

**预期结果**:

**验收标准**: 加载时间≤3秒且所有元素正常显示

**测试目标**: 验证首页导航功能正确性

**预期结果**: 导航跳转正确，无页面错误

**验收标准**: 100%导航成功率

### TC-S01-03: 移动端首页适配 [P1]
**测试目标**: 验证移动端首页适配效果  
**测试环境**: iPhone/iPad/Android设备  
**测试步骤**:
1. 在不同设备上访问首页

3. 测试触摸交互功能

**预期结果**:

**验收标准**: 多设备100%适配

## 📝 TC-S02: 新建报价功能

### TC-S02-01: 客户信息录入验证 [P0]
**测试目标**: 验证客户信息录入字段校验  
**测试步骤**:
1. 进入新建报价页面
2. 测试客户姓名字段：
   - 输入空值
   - 输入21个字符
   - 输入20个字符  
   - 输入特殊字符
3. 测试电话号码字段：
   - 输入空值
   - 输入10位数字
   - 输入12位数字
   - 输入11位数字
   - 输入含字母的内容
4. 测试备注字段：
   - 输入101个字符
   - 输入100个字符

**预期结果**:
- 姓名：空值报错，>20字符报错，≤20字符通过
- 电话：空值报错，非11位数字报错，11位数字通过
- 备注：>100字符报错，≤100字符通过
- 实时校验响应时间≤0.2秒

**验收标准**: 所有校验规则100%准确执行

### TC-S02-02: 产品选择与价格计算 [P0]
**测试目标**: 验证产品选择和价格自动计算功能  
**测试步骤**:
1. 选择单个产品，数量为1
2. 验证单价和总价显示
3. 修改数量为5，验证价格更新
4. 选择多个产品，验证价格累计
5. 选择地毯类产品，测试多SKU选择
6. 删除产品项，验证价格重新计算

**预期结果**:
- 价格计算准确率≥99.9%
- 计算误差≤0.01%
- 价格实时更新响应时间≤0.3秒
- 地毯多SKU规格选择正常

**验收标准**: 价格计算100%准确，实时更新正常

### TC-S02-03: 图片上传功能 [P1]

- 支持JPG/PNG/GIF格式
- 单个文件<5MB限制生效
- 非图片文件上传被阻止
- 图片预览正常显示
- 删除功能正常

**验收标准**: 图片上传功能100%正常，安全验证有效

### TC-S02-04: 报价单生成性能 [P0]
